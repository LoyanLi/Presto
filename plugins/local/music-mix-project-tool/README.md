# Music Mix Project Tool

Local non-official Presto tool plugin for creating music mix project folders.

## Install

Use the app's `Install Local Directory` flow and select `plugins/local/music-mix-project-tool/`.

## Behavior

- Remembers one default base root in plugin-local storage
- Allows changing the base root on each run
- Creates project folders using `YYMMDD_歌名`
- Creates only the selected top-level sections
- Creates standard document files inside `04_Documents`
