# Presto 0.3.3

Presto `0.3.3` 是当前 `0.3.3` 线的正式发布，直接合并了原先计划放在 `0.3.3-alpha.1` 中的更新检查、Python runtime 便携化修复，以及本轮安装包体积优化。

## 本版重点

- 应用启动时会主动检查 GitHub release，并根据用户设置决定是否包含预览版。
- 宿主发现新版本后会弹出更新提示，用户可以直接跳转到 release 页面。
- 打包态 bundled Python runtime 不再外链用户机器本地 `/Library/Frameworks/Python.framework`。
- bundled `Python.framework`、`Python.app` 启动器、helper script 与 `pyvenv.cfg` 都已经收口为随包分发的自包含形态。
- 正式安装包额外裁掉 Python 标准库中的测试、开发和桌面 GUI 冗余内容，显著缩小 `.app` 与 `.dmg` 体积。
- 应用、workspace package、Tauri 与 FastAPI 版本基线统一到 `0.3.3`。

## 升级说明

- 当前 release 检查源仍然是 GitHub Releases。
- 当前 updater 仍然是“发现新版本并引导跳转下载”，不是增量补丁分发。
- 当前实际支持的 DAW 目标仍然只有 `pro_tools`。
- `0.3.3-alpha.1` 没有单独发布，正式用户应直接使用 `0.3.3` 产物。

- `x64` App：`release/tauri/x64/Presto.app`
