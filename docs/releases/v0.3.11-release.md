# Presto 0.3.11

`0.3.11` 更新应用与插件协议相关的版本基线，并把 PTSL 命令最低宿主版本需求收口到统一的 `minimumHostVersion` 语义。

## 重点变化

- 应用、workspace package、Tauri、Rust crate、contracts 与 backend 版本基线统一到 `0.3.11`。
- PTSL catalog 不再向外暴露 `introducedVersion`，统一使用 `minimumHostVersion`。
- 插件宿主现在可以根据插件 `requiredCapabilities` 自动读取 DAW capability snapshot 中的 `minimumHostVersion`，判断当前 Pro Tools/PTSL 版本是否满足插件需求。
- 官方插件 manifest 移除手写 capability version requirement，改由 host 根据 capability 自动推导最低宿主版本需求。

## 安装包

`0.3.11` 安装包尚未在本次版本号更新中生成。生成后在这里补充 `arm64` / `x64` DMG 文件名、SHA-256 与文件大小。
