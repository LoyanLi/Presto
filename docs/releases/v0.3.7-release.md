# Presto 0.3.7

`0.3.7` 是当前正式发布版本。这一版把宿主里的运行统计收口成独立的 `Runs` 页面，并把统计口径、榜单展示和页面设计语言统一到当前 host shell。

## Highlights

- Host shell 新增独立 `Runs` 页面，围绕 `workflow`、`automation`、`command` 三个维度展示成功运行排行。
- `workflow` 榜改为只统计成功完成的 workflow job；`workflow.run.start` 不再计入 workflow 榜，但仍然计入 command 榜。
- 成功 workflow job 会把内部每次成功执行的 command 一并沉淀到运行统计里，因此一次 import workflow 可以在 command 榜累计多次 run，而不是只记一次 workflow 触发。
- command 榜支持宿主侧文字译名显示，不再直接暴露原始 capability / command ID。
- `Runs` 页面改为与现有 host shell 完全一致的设计语言：同一套配色、卡片体系、间距和交互；宽屏固定三列，窄屏退回单列，滚动只发生在各卡片内部。

## Upgrade Notes

- 根 `package.json` 仍然是唯一手工维护的产品版本源；workspace package、Tauri 配置、Rust crate、contracts 版本常量和 backend 版本常量继续通过同步脚本派生。
- `Runs` 页面当前只统计成功执行；失败执行不会进入排行结果。
- command 榜现在反映实际成功调用次数，而不是 workflow 入口次数；如果一个 workflow 内部串行调用多个 command，这些 command 会分别累计。
