# Presto 0.3.2-1

Presto `0.3.2-1` 是对 `0.3.2` 的发布链收口版本，重点修复打包态 bundled Python runtime 仍然外链构建机 Python framework 的问题。

## 亮点

### Bundled Python runtime 改为真正随包分发

- bundled `python3` 不再继续依赖用户机器上的 `/Library/Frameworks/Python.framework/Versions/3.13/Python`
- 打包资源现在会一并带上所需 `Python.framework` 运行时文件
- 重写后的解释器会在资源目录内解析自己的 Python 动态库，不再要求用户预装同版本 Python framework

### Helper script 与运行时配置去除构建机绝对路径

- `fastapi` / `uvicorn` helper script 不再保留本地 `python.staging` shebang
- `pyvenv.cfg` 不再写入构建机 staging 路径
- 发布测试新增对 bundled Python 可移植性的校验，避免同类问题再次进入安装包

## 升级说明

- 当前正式目标 DAW 仍然是 `Pro Tools`
- 当前正式发布形式仍然是完整 `.app` 与 `.dmg` 安装包
- `0.3.2-1` 不引入增量安装包机制；当前仓库也没有 Sparkle / Tauri updater delta / appcast 等增量发布基础设施
- App、workspace package、Tauri 与 FastAPI 版本基线统一到 `0.3.2-1`
