# Presto v0.2.0 Release Notes

Release date: 2026-03-14
Tag: `v0.2.0`

## 1. 版本概览

v0.2.0 聚焦三件事：
- 可控性：新增全局设置与开发者模式开关，普通音乐人默认不暴露开发配置。
- 稳定性：导入/导出运行时守护、错误语义统一、阶段状态和 ETA 对齐。
- 可见性：导出支持手机扫码只读查看进度，并提供 ETA 与预计完成时间。

## 2. 面向用户的主要更新

1. 新增全局设置页（含开发者模式控制）。
2. 新增开发者页（诊断、日志导出、报错测试器，本地语言适配）。
3. 导入流程支持更稳定的批次执行与取消处理。
4. 导出流程支持手机扫码查看进度（只读、临时链接、可手动关闭）。
5. 导出 ETA 展示优化：
   - 第一个 snapshot 显示“正在计算”。
   - 后续阶段平滑推进，避免跳动感过强。

## 3. 安装包

- Apple Silicon: `frontend/release/Presto-0.2.0-arm64.dmg`
- Intel: `frontend/release/Presto-0.2.0-x64.dmg`

建议发布时分别提供两条下载项，用户按机型选择，减少无效下载体积。

## 5. Release 文案（可直接使用）

### 5.1 短文案（社媒/群公告）

Presto v0.2.0 已发布。  
本次重点升级全局设置与开发者模式、导入导出稳定性、友好错误提示，以及“手机扫码只读查看导出进度”。欢迎升级体验。

### 5.2 标准文案（GitHub Release）

Presto v0.2.0 brings a stability-focused workflow upgrade for Pro Tools users.

Highlights:
- Global Settings and guarded Developer Mode.
- More reliable import/export runtime behavior and clearer progress semantics.
- Localized friendly error UX across core flows.
- Mobile QR read-only export progress with temporary session links.
- Smoother ETA behavior, including “calculating” state for early-stage export.

Installers:
- macOS arm64
- macOS x64

Please choose the installer that matches your Mac architecture.
